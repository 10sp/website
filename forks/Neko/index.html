# Neko

# About

This is a MangaDex specific fork of Tachiyomi and TachiyomiJ2K. This contains features specific to MangaDex that those versions cannot offer. It also features almost every feature that J2K has except for extensions (aka other sources), custom covers, and local manga.

# Screenshots

# Mangadex specific features:

  • Supports native login to MangaDex and supports 2FA
  • MDList support
  • Similar Manga recommendations
  • Manually sync of Manga to and from MangaDex follows list
  • Ability to filter chapters and skip by scanlator group
